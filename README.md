# 🤖 DSA Bro

DSA Bro is an AI-powered Data Structures and Algorithms tutor built using Node.js, Express.js, Google Gemini API, HTML, CSS, and JavaScript.

It helps students understand DSA concepts, analyze algorithm complexity, and learn through an interactive conversational interface.

## ✨ Features

* AI-powered DSA tutoring using Google Gemini
* Conversation memory
* Interactive chat interface
* DSA-focused responses through custom system instructions
* Markdown-rendered responses
* Code block support
* Suggested topic prompts
* Typing indicator
* Modern dark-themed UI

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### AI Integration

* Google Gemini API
